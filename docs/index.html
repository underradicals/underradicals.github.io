What is D2 ​

D2 Data is an information system built on top of the Destiny2 Manifest and all the data it provides. The pages found in this documentation site provided by vitepress cover setup, tools, patterns, and white papers used in the development of not only the code but also the project.

This system is comprised of 11 projects: Each with its corresponding Backend and Frontend projects.
