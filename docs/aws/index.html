Intro to the Cloud ​

Intended Audience ​

This learning plan is intended for individuals who meet the following conditions:

  1. Have up to 6 months of exposure to AWS Cloud design, implementation, and/or operations;

  2. Are from non-IT backgrounds or in the early stages of pursuing an AWS Cloud career or work with people in AWS Cloud roles; and

  3. Are preparing for the AWS Certified Cloud Practitioner (CLF-C02) exam.

4-Step Plan ​

From start to certified: Prepare for your exam with confidence by following this 4-step plan.

At the end of this course, you will be able to implement the 4-step exam prep plan to structure your AWS Certification exam preparation.

Step 1 ​

Time commitment: Up to 6 hours We recommend covering this material in weeks 1 and 2 to provide time to review the exam topics and to identify gaps in knowledge that might require further study.

  1. Review the exam details page(opens in a new tab).

  2. Review the exam guide (available on the exam details page).

  3. Take the AWS Certified Cloud Practitioner Official Practice Question Set to understand exam-style questions.

  4. Take the AWS Certified Cloud Practitioner Official Pretest.

Step 2 ​

Time commitment: varies We recommend covering this material in weeks 2 to 4.

To refresh your knowledge and skills, do the following:

  • Review the resources listed.

  • Take the recommended AWS Skill Builder digital courses if you feel you need to refresh your AWS knowledge and skills. Take as many courses as you need.

  • These courses are also included at the end of the courses in the Outline tab for this exam prep plan, but are not required to complete this exam prep plan.

Digital courses ​

Take AWS Cloud Practitioner Essentials. (7 hours)

  • Take AWS Technical Essentials, modules 1, 2, 3, 5, and 6. (4 hours)

  • Take Getting Started with AWS Cloud Essentials, focusing on these lessons: (1 hour)

    • Understanding the AWS Global Infrastructure
    • Core Services Overview: Storage
    • Core Services Overview: Pricing

Game-based Learning ​

  • Gain hands-on experience using AWS services by playing AWS Cloud Quest: Cloud Practitioner. (time varies)

Step 3 ​

Time commitment: up to 11 hours

For each domain, we recommend the following steps:

  1. Take the corresponding Domain Review course. Explore topics and how they align to AWS services and to specific areas of study.

  2. Take the corresponding Domain Practice course. Reinforce your knowledge and identify learning gaps with exam-style questions and flashcards.

  3. Complete AWS Builder Labs and AWS SimuLearn Challenges for additional practice, when available.

  4. Review the additional resources provided at the end of each exam domain.

AWS Escape Room ​

After you have completed courses for all domains, continue preparing with AWS Escape Room: Exam Prep.

Optional ​

For a different approach to preparing for the exam, check out a future AWS Power Hour(opens in a new tab) series on AWS Certified Cloud Practitioner. [Filter for “Virtual” and “AWS Power Hour (Exam Prep)”.] The lessons will overlap with the videos in the exam prep course.

Step 4 ​

Time commitment: 2.5 hours

We recommend covering this material in week 5.

  1. Take the Official Practice Exam to simulate the actual exam. You can take the practice exam timed. This is the default setting. You will not see the correct answers until you finish and submit the practice exam.

  2. Review your score report thoroughly.

  3. Review the questions and the linked documentation provided for correct and incorrect answer options in each question.

  4. Go back to the resources in Step 2 if needed.
